# C++ Folder Structure
It is simple C++ project folder structure.
### Example folder structure
```
.
├── bin
│   └── project
├── build
│   ├── main.o
│   └── student.o
├── doc
├── include
│   └── student.hpp
├── lib
├── Makefile
├── README.md
└── src
    ├── main.cpp
    └── student.cpp
```
## Start
``` make start```
## Only Compile
``` make compile```
## Only Link to Object
``` make link```
## Run Executable File(s)
``` make run```
## Remove All Executable Files and Object Files
``` make remove```
